# 🤖 Leandro Lanzillo

**`Cientista de Dados`**

Me chamo Leandro, sou estudante de Ciência de Dados na FIAP e apaixonado por transformar dados em insights estratégicos e tomada de decisão orientada por evidências.

Tenho experiência com Python, SQL e bancos de dados, além de interesse em análise de dados, modelagem, visualização e aplicações de Data Science no mercado financeiro.

Profissionalmente, atuei no Itaú Unibanco, onde trabalhei com análise de perfis de clientes, CRM e projeção de resultados para gestão de carteira, desenvolvendo uma visão orientada a dados para tomada de decisões e geração de valor.

Também possuo certificações do mercado financeiro (CEA, CPA-20 e ANCORD) e conhecimento em análise de investimentos, alocação de ativos e análise macroeconômica, o que reforça meu interesse na interseção entre dados, finanças e estratégia.

Aqui no GitHub compartilho projetos, estudos e experimentos em dados, programação e analytics.
